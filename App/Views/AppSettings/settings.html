Import Website Content

Add content to your website, including HTML, CSS, LESS, & Javascript. Make sure the zip file you upload contains files within the following folder structure. Include at least one of the folders below within your zip file.

  • wwwroot/**
    Any files or sub folders located within wwwroot will be copied over.
  • content/pages
    Each web page should contain the following files: {page}.html, {page}.less, {page}.js, {page}.json, and {page}_{lang}.json. Replace {page} with the name of your page, such as blog.html. Also, replace {lang} with your preferred language, such as "en" (English).
  • content/partials
    Each partial should contain a html & less file.
  • css/website.less
    You are able to overwrite the site-wide LESS file if it is required.
Export Website Content

Download a zip file containing all of the HTML, CSS, LESS, & Javascript content belonging to this website.

Reset Website Cache

This will make sure that all cached web pages are flushed from server memory and reloaded from disk.

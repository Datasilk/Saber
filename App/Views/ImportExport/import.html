Add content to your website, including HTML, CSS, LESS, & Javascript. Make sure the zip file you upload contains files within the following folder structure. Include at least one of the folders below within your zip file, or simply import a zip file that was exported from Saber.

  • wwwroot/**

    Any files or sub folders located within wwwroot will be copied over.

  • content/pages

    Each web page should contain the following files: {page}.html, {page}.less, {page}.js, {page}.json, and {page}_{lang}.json. Replace {page} with the name of your page, such as blog.html. Also, replace {lang} with your preferred language, such as "en" (English) or "es" (Spanish).

  • content/partials

    Each partial should contain a html & less file.

  • content/website.less

    You are able to overwrite the site-wide LESS file if it is required.

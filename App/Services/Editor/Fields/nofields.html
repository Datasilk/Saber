To add content, create special variables within the file {{filename}}.html.

For example, the following HTML code will generate two content fields:

<div>written by {{author-name}} on {{publish-date}}</div>

Keep in mind the syntax rules for special variables.

  • They must begin with {{ and end with }}
  • They must be alpha-numeric and lowercase, and can contain dashes - and underscores _
  • There can be no spaces within a special variable
